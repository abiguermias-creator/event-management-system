\# Event Management System



A web-based Event Management System built with Node.js, Express, EJS, and PostgreSQL.



\## Features



\* Create events

\* View event details

\* Edit events

\* Delete events

\* PostgreSQL database integration

\* Responsive UI with EJS templates



\## Technologies Used



\* Node.js

\* Express.js

\* PostgreSQL

\* EJS

\* CSS



\## Installation



1\. Clone the repository:



```bash

git clone https://github.com/abiguermias-creator/event-management-system.git

```



2\. Install dependencies:



```bash

npm install

```



3\. Configure the database in `db.js`.



4\. Start the server:



```bash

node server.js

```



5\. Open:



```

http://localhost:3000

```
